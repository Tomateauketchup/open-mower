# Brainstorming


Contraintes :

* Le robot doit résister aux intempéries (pluie, vent).
* Pas d'utilisation de fil guide.
* Le robot doit avoir une bonne motricité sur des sols de type sous-bois.
* La couverture de tonde doit pouvoir atteindre 5000 m2.
* Dans les premières versions, il n'est pas prévu que le robot puisse tondre les bordures avec précision.
* Les surfaces cibles sont de type jardin, parc, stade de sport.
* Le projet doit être conçu avec l'objectif de réduire les couts de fabrication et de maintenance au minimum.
* L'utilisateur doit être dans la capacité de réparer son robot lui-même et de pouvoir acheter des pièces de remplacement facilement à moindre cout.
